# DimOS Interactive Viewer

`uv run dimos --viewer rerun-web run unitree-go2` で起動する Web UI の解説。

## 画面構成

`http://localhost:7779/` を開くと左右2ペイン構成のダッシュボードが表示される。

```
┌────────────────────────┬──┬──────────────────────────────────┐
│   Command Center       │  │      Rerun Viewer                │
│   (左パネル ~30vw)      │  │      (右パネル、残り幅)            │
│                        │  │                                  │
│  コストマップ (SVG)      │  │  Camera (1/3) │  3D View (2/3)  │
│                        │  │                                  │
│  ────────────────────  │  │  ↕ タイムラインバー               │
│  [ボタン群]             │  │                                  │
└────────────────────────┴──┴──────────────────────────────────┘
```

## 左パネル：Command Center

### メイン表示エリア（黒背景 SVG）

OccupancyGrid を D3.js の `interpolateTurbo` カラースケールで色付けした **コストマップ（ヒートマップ）**。

| 色 | 意味 |
|-----|------|
| 黒 | 未知領域 |
| 暗紫（`#2d2136`） | 未探索 |
| 黄〜赤 | 障害物に近い高コスト領域 |

- **ナビゲーションパス** — A* が計算した経路を線で表示
- **ロボット位置** — 現在位置をマーカーで表示
- **クリックでゴール指定** — マップ上をクリックするとその座標にロボットが向かう

### 下部ボタン群

| ボタン | 動作 |
|--------|------|
| **Use GPS** | 左パネルを Leaflet 地図に切替。GPS 座標（緯度・経度）でゴールをキューに追加 |
| **Use Costmap** | コストマップビューに戻る（Use GPS 後に表示） |
| **Start Exploration** | ウェイブフロントフロンティア探索を開始。自律的に未探索エリアを走り回りマップを拡張 |
| **Stop Exploration** | 探索を停止（ボタンが赤になりアクティブ状態を示す） |
| **Go Home** | ワールド原点 (0, 0) へナビゲート |
| **Stop** | ロボットの現在位置をゴールに送信してその場で停止 |
| **Start Keyboard Control** | キーボード手動操縦を有効化（10 Hz でコマンド送信） |

#### キーボード操縦キーマップ

| キー | 動作 |
|------|------|
| `W` / `S` | 前進 / 後退（linear.x ±0.5 m/s） |
| `A` / `D` | 左旋回 / 右旋回（angular.z ±0.8 rad/s） |
| `←` / `→` | 左横移動 / 右横移動（linear.y） |
| `↑` / `↓` | ピッチ上 / 下（angular.y） |
| `Space` | 即時停止（全速度ゼロ） |
| `Shift` + 移動キー | **Boost モード**（速度 2倍） |
| `Ctrl` + 移動キー | **Slow モード**（速度 0.5倍） |

ウィンドウのフォーカスが外れると自動で無効化される。

## 右パネル：Rerun Viewer（ポート 9090）

`unitree_go2_basic.py` の `_go2_rerun_blueprint()` で定義したレイアウト：

| ペイン | 表示内容 |
|--------|----------|
| **Camera（左 1/3）** | ロボットのカメラ映像（2D） |
| **3D View（右 2/3）** | ボクセルマップ、コストマップオーバーレイ、TF フレーム、ロボット形状ボックス（ワイヤーフレーム） |

### タイムラインバー（Camera・3D の下）

Rerun ビューアー組み込みの標準 UI。DimOS 側のカスタム実装ではない。

- **時系列スクラブ** — バー上のカーソルを左右にドラッグして過去データの任意時点を確認
- **ストリームごとのトラック** — カメラ映像・PointCloud・TF・コストマップなど各ストリームが別レーンで表示
- **再生コントロール** — 再生・一時停止・速度変更
- **ライブモード** — 右端に追従している状態がリアルタイム表示。スクラブすると過去データ参照モードへ移行
- セッション中のデータはメモリ上限（デフォルト `auto`）まで蓄積され、超過すると古いものから自動削除

## 起動コマンド

```bash
# デフォルト（Rerun デスクトップ）
uv run dimos run unitree-go2

# Web UI（ブラウザ自動起動）
uv run dimos --viewer rerun-web run unitree-go2

# Foxglove
uv run dimos --viewer foxglove run unitree-go2

# ヘッドレス
uv run dimos --viewer none run unitree-go2

# メモリ上限を指定
uv run dimos --memory-limit 4GB run unitree-go2
```

`--viewer` などのグローバルオプションはサブコマンド（`run`）の**前**に指定する。
