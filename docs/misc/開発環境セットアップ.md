# Claude Code
- 権限
	- `.claude/settings.json`に書く。チーム共有
	- 個人的な設定は `.claude/settings.local.json`に書く。これはgitignoreする
- superpowers
	- https://qiita.com/nogataka/items/c2e73515e65533986421
	- AI駆動開発にワークフローをもたせるフレームワーク
	- `brainsotming` -> `writing-plans` -> `TDD` -> `review`
	- 勝手にsubagentへタスクを振り分けてくれる。コンテキストが膨張するリスクが下がる
	- タスクの難易度に応じてモデルを使い分けてくれる
	- git worktreeも内包していて、並列作業できる
	- 設計を`docs/`に保存することで後追い・チーム共有できる
- language
	- /config -> language -> Japanese
	- /voiceで音声入力できる
- 便利コマンド
	- /resume
	- /branch
	- /btw
		- シングルターンのみ。呼んだらesc押すと戻る

# Obsidian (ドキュメント管理)
- `docs/`に`.obsidian`を配置することで、obsidian vaultとして扱う。
- ソースコードの近くに置くことで、開発中Claude Codeに「`docs/xxx`に書いておいて」と言える
- ObsidianのUI上でもClaudeを使える (Claudian plugin)
