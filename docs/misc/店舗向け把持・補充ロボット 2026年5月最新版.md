# 店舗(コンビニ・スーパー)向け把持・補充ロボット技術調査レポート（2026年5月時点）

## TL;DR

- 店舗での「把持→移動→配置」自動化は、PETボトル・缶など**形状の揃った定型商品**に限れば既に商用フェーズに到達した（Telexistence TX SCARA / TX Ghost が日本のファミリーマート約300店舗とセブン-イレブン一部店舗、ローソン2店舗で稼働、自律成功率約98%）が、おにぎり・弁当・袋菓子・生鮮など**変形物・滑り物・透明物**を含む多品種棚への汎用展開は2026年5月時点で実証実験段階にとどまる。
- 技術的ブレークスルーの中心は **VLA（Vision-Language-Action）基盤モデル**で、Physical Intelligence π0/π0.5/π0-FAST、Google DeepMind Gemini Robotics 1.5、NVIDIA Isaac GR00T N1、Figure AI Helix/Helix 02 が2024–2025年に相次いで発表され、未知物体への zero-shot 把持・長期タスク・身体性転移（cross-embodiment）を実現しつつあるが、業務レベルの信頼性（>99%）には未達。
- **店舗導入のボトルネックは技術より経済性**：Walmart は2020年11月に約500店舗で稼働していた Bossa Nova 棚スキャンロボットの契約を打ち切り（展開予定だった1,000店舗の半分にとどまった段階）、「人間でも同等の結果が得られる」と評価。一方、人手不足が深刻な日本ではセブン-イレブン×Telexistence「Astra」humanoid を2029年商用展開目標、KDDI×ローソンが2025年11月から VLA 搭載アーム（2指+5指）で売場品出し実証を開始するなど、社会実装速度では日本が先行している。

## Key Findings（2026年5月時点の現在地）

## Details

### 1. 把持計画・ハンド設計

#### 1.1 把持戦略の二極化：吸着+並列指 vs マルチフィンガー

店舗バックヤード/売場で実商用化されているハンドは大別して2系統である。

- **吸着+並列指ハイブリッド（TX SCARA/TX Ghost 等）**：PETボトル・缶など円筒形・剛体表面に最適。Telexistence の TX SCARA は専用AI「Gordon」で1日1,000本の補充を自律成功率約98%で実行（NVIDIA Jetson プラットフォーム搭載）。ファミリーマート約300店舗に量産展開済。
- **ソフトグリッパ（食品グレード）**：Soft Robotics Inc. の mGrip 製品ラインは2024年8月に独 Schmalz が買収し、IP69K防水（高圧洗浄対応）+食品グレード材料で、ベーカリー・タンパク質・果物・野菜のピックを実現。20mm まで対応のコンパクトフィンガーが追加された。

#### 1.2 ソフトグリッパ最新動向

- **ユニバーサル・ジャミンググリッパ**：Cornell/iRobot 由来の granular jamming（Brown et al., PNAS 2010）は、エンベロープ+粒状材料に真空負荷で対象物に密着・硬化する仕組み。商用化を試みた Empire Robotics（VERSABALL®）は2016年に事業停止しており、フラット物体や柔軟物体への適応に限界。改良研究は継続（MDPI Machines 2024: silicone/latex × glass beads サイズの組合せ最適化）。
- **Festo Bionic Learning Network**：BionicSoftHand 2.0 を ETH チューリッヒ協働の BionicMobileAssistant に統合（空気圧+触覚グローブ）。商用ラインでは FinRay 効果フィンガ DHAS、カメレオン舌型 DHEF が量産販売中。
- **ロボット食品向けハンドの市場**：Global Market Insights によればソフトグリッパ市場は2024年6.78億ドル→2034年12.8B USD（CAGR 35.1%）と予測（マーケティング予測のため割引解釈が必要）。

#### 1.3 触覚センサ

- **GelSight Mini**（2022年商用化）とその派生 **DIGIT**（Meta AI 共同開発、低価格・小型で多指ハンド搭載可能）が研究の主流。エラストマー変形を内部カメラで撮影し CNN で力推定・滑り検知。
- DIGIT は USB接続・PyTorch エコシステム互換でほぼ「標準触覚センサ」となった。Figure 03（2025年10月発表）は「クリップ1個の重さを検知する」触覚センシングを内蔵すると主張。
- 触覚+音響融合の最新研究として「Resonance Fields」（2024–2025）が 47種類の家庭用物体で 94.2% の把持安定性分類精度を報告。

#### 1.4 把持点検出（Grasp Detection）の現状

- **GraspNet-1Billion / AnyGrasp**（上海交通大 Fang らによる）：190 cluttered シーン+97,280画像、自己教師あり学習でほぼ「人間レベル」の汎用把持を実現（IEEE T-RO 2023）。2025年11月時点でも CUDA 12.8 / Python 3.13 対応の SDK 更新が継続。
- **Contact-GraspNet**（NVIDIA）：1700万 simulated grasp で訓練、AnyPart 評価では単体物体 65.0% / クラッタ 60.8% の成功率を達成、AnyGrasp（29.7% / 20.3%）を上回るとの報告も（arXiv 2406.05951）。
- **GraspGen**（NVIDIA, 2025）：diffusion ベースの 6-DoF 把持生成。FetchBench で SOTA、ACRONYM データセットを使用。
- **言語条件付き部位把持**：AnyPart など open-vocabulary detection + 部位 segmentation + 6-DoF grasp を統合する流れが2024年以降の主流。

### 2. モーション計画・障害物回避

#### 2.1 古典手法と学習手法のハイブリッド化

- 棚内狭隘空間への進入では、商品位置が「既知」でも棚板/隣接商品との非接触保証が必要。サンプリングベース（RRT*, BiRRT）+最適化ベース（CHOMP, TrajOpt）+ collision-aware ニューラル軌道生成が併用される。
- **GraspGen の指摘（arXiv 2507.13097）**：把持成功率の上限は「衝突なしで引き抜ける経路が存在するか」に大きく依存し、把持点が良好でも retraction 経路が無いと失敗。これが実店舗シェルフでの最大の隠れた制約。

#### 2.2 リアクティブ制御

- **π0（Physical Intelligence）**：flow matching ベースで 50Hz のスムーズな連続軌道生成、リアクティブ制御に必要な低レイテンシを実現。
- **Gemini Robotics 1.5（2025年9月）**：「observation→action」を250ms、有効制御周波数 50Hz（クラウド+エッジ二層構造）。Pixmo-Point ベンチで pointing 精度 25.8%→49.5%、SUN-RGBD 3D detection 30.7%→48.3%。

#### 2.3 モバイルマニピュレータの全身協調

- **KDDI×ローソン実証**（2025年11月8日開始、ローソン S KDDI高輪本社店）：自律走行台車+アーム+VLA で「棚奥への商品補充」を試行。**2指グリッパーモデルと5指ハンドモデルの両者を併用**して適性を比較する点が技術的にも特徴的。
- **Helix 02**（Figure AI, 2025–2026）は単一ニューラルネットで歩行・バランス・両手操作を統合する「whole-body loco-manipulation」を実現。4分間のフルキッチン食洗機装填デモを公開。

### 3. VLA等のAIモデル・学習手法

#### 3.1 VLA基盤モデルの系譜と2025年の収束

- **RT-1 / RT-2 / RT-X**（Google, 2023–2024）：Open X-Embodiment（22ロボット、21機関、527スキル、160,266タスク）。RT-1-X が既存単機ベースラインを+50%、RT-2-X が +3× の一般化向上。
- **Physical Intelligence の π系**（2024–2025）：
  - π0（2024年10月）：7プラットフォーム×68タスクで学習、flow matching + 50Hz 制御。
  - π0-FAST（2025）：autoregressive action tokenizer で訓練5倍高速化。
  - π0.5（arXiv 2504.16054, 2025年4月）：異種データ（他ロボット、人ビデオ、Webデータ、言語サブタスク予測）co-training で完全新規家庭（訓練データ未収録のキッチン・寝室）で汎化を実証。
- **NVIDIA Isaac GR00T N1**（GTC 2025年3月発表、arXiv 2503.14734）：dual-system（VLM = NVIDIA-Eagle+SmolLM-1.7B / DiT action expert）、Fourier GR-1・1X Neo・Agility Digit など多 embodiment で動作。Isaac GR00T Blueprint が 780,000合成軌道（=6,500時間=9ヶ月分の人間デモ相当）を11時間で生成、実データ+合成で +40% 性能。
- **Google DeepMind Gemini Robotics 1.5 / ER 1.5**（2025年9月25日発表）：VLA（1.5）+ embodied reasoning VLM（ER 1.5）の2階層エージェント。ALOHA 2 で学習したスキルが追加微調整なしで Franka・Apptronik Apollo に転移（cross-embodiment 学習）。ER 1.5 は Google AI Studio から API 公開、1.5 は select partners のみ。
- **Figure AI Helix / Helix 02**（2025年2月発表）：Figure 内製 VLA、商用低消費電力組込 GPU 上でローカル実行可能と主張。Helix 02 は単一 NN による whole-body 統合。Brookfield と提携し、>1兆ドル AUM の住宅・オフィス・物流空間で人間 ego-centric 動画を収集して事前学習データセット構築中（2025年9月発表）。

#### 3.2 模倣学習・拡散モデル系

- **Diffusion Policy**（Chi et al., RSS 2023→IJRR 2024）：条件付き拡散による action sequence 生成、マルチモーダル分布表現が強み。FiLM 条件付けで観測条件を注入する5要素（observation seq, action seq, receding horizon, U-Net/Transformer, FiLM）が現代の標準。
- **3D Diffusion Policy（DP3）**（RSS 2024）：点群+拡散ポリシーで 72 simulation タスクを10デモで、4実機タスクを 40デモで 85% 成功。
- **ACT**（Action Chunking Transformer, ALOHA 系）や **DTP**（Diffusion Trajectory-guided Policy, 2024）も長期タスクの主要選択肢。

#### 3.3 データ収集・Sim-to-Real

- **Open X-Embodiment**（Hugging Face / RLDS フォーマット）が de facto 標準。Telexistence は2026年1月から「Robotic Motion Dataset Generation Service」を商用開始予定で、TX Ghost の店舗実稼働データを商品化。
- **合成データ**：NVIDIA Isaac Sim / Isaac Lab / Cosmos、Google DeepMind+NVIDIA+Disney Research 共同開発の Newton 物理エンジン、MuJoCo-Warp（>70倍高速化）が2025年に発表。

#### 3.4 汎化性能の現実

- **Helix のロジスティクス報告**：訓練デモ 10時間→60時間で、パッケージ処理時間 6.84s→4.31s（58%スループット向上）、バーコード成功率 88.2%→94.4%（Figure AI 公式 2025年）。
- **二腕 nonprehensile グロサリー把持の学術ベンチ（arXiv 2409.15465）**：複製スーパー環境で uncluttered 90% / cluttered 67% 成功率（102/45 trials）。
- π0.5/GR00T N1/Helix のいずれも、グロサリー特化の「未知物体 X% 把持」という単一クリーンな数値は公式に公表されていない（タスク進行率や定性デモが中心）。**現状でクリアな業務 SLA（>99%）を保証する VLA は無い。**

### 4. 社会実装事例（2026年5月時点）

#### 4.1 日本国内

**Telexistence（TX）**
- TX SCARA：ファミリーマート約300店舗で2022年8月から量産配備、バックヤード飲料補充自動化、自律98%/2%は VR テレオペレーターによる遠隔リカバリ。
- TX Ghost（後継機）：2025年9月、7-Eleven 東京店舗で実稼働開始。スマートシェルフ（デジタル在庫棚）と連動。同年6月にローソン2店舗（高輪ゲートウェイシティ店、S KDDI高輪本社店）にも導入。
- Physical Intelligence と2025年6月提携：π系基盤モデルで「ロールオーバーした飲料の復帰」など難リカバリ手順を自律化することを目指す。
- Astra（2029年商用化目標、2025年9月発表）：セブン-イレブン日本（SEJ）約20,000店舗のデータと組合せ、VLA 基盤モデル搭載 humanoid。AIRoA（早稲田大・松尾豊・トヨタなど）とデータ生成で連携。
- 2025年8月 Japan Startup Awards 内閣総理大臣賞受賞。2026年1月から Robotic Motion Dataset Generation Service 開始。

**KDDI×ローソン**（2025年11月8日開始、「ローソン S KDDI高輪本社店」）：VLA を活用、自律走行台車+アーム（2指/5指）で売場の菓子・インスタント食品まで品出し。AI 棚スキャン+人流分析+棚割最適化の統合実証。2030年度に店舗オペレーション30%削減目標。

**ローソン Model T 竹芝店**（2020年9月開店、NEDO支援）：Telexistence Model-T による遠隔操作→段階的自律化。日本がコンビニ補充ロボットの世界的先行例となった起点。

#### 4.2 海外

- **Walmart × Bossa Nova**：2020年11月に契約終了。約500台が約500店舗に展開済（展開予定1,000店舗の半分時点）で打ち切り。WSJ/CNBC によると「人間でも同等」「シンプル」「顧客の反応への懸念」「パンデミックでオンライン注文ピッカーが既に売場を歩いている」などが理由。教訓として、shelf-scan のみで補充は人がやる構図ではコスト効果が出にくいことが明確に。
- **Simbe Robotics Tally**：2025年11月時点で導入10年。Fortune 100含む10ヶ国・3大陸で稼働。累計 600M shelf gap 検出、18B 価格タグスキャン、180万km走行、44.8B 棚画像。Tally 4.0（2025年）：12時間稼働、NVIDIA edge AI、3D depth+360°魚眼。BJ's Wholesale Club、Harmons（17店舗チェーン全展開、欠品20%削減）、HomeBase USA、SpartanNash、Wakefern ShopRite 等。
- **Ahold Delhaize Albert（チェコ）**：Brain Corp プラットフォームで AI 棚スキャンロボット展開（2024–2025）。米 Ahold Delhaize の Stop & Shop / Giant では Badger Technologies の「Marty」が300店舗以上で安全監視+棚スキャン。
- **Agility Robotics Digit × GXO**：2024年6月から RaaS 契約、2025年11月時点で10万トート以上を実商用搬送。SPANX の Atlanta 倉庫で稼働、Apollo・Stretch とともに humanoid 倉庫導入のリファレンスケース。
- **Apptronik Apollo**：Mercedes-Benz（2024年）+GXO（2024–2025）+Jabil（2025年）とパイロット。2025年に Series A 4.15億ドル（B Capital と Capital Factory が共同リード、Google 参画）、2026年2月に追加 Series A-X 5.2億ドルを B Capital と Google が共同リード（AT&T Ventures、John Deere、Qatar Investment Authority 等参画）で総額約9.35億ドル、valuation 約50億ドル。Gemini Robotics 1.5 を統合。
- **Figure AI**：Figure 02 を2024年12月から有償顧客に納入。BMW で自動車製造ライン稼働。2025年9月 Series C 10億ドル超（Parkway Venture Capital がリード、Brookfield Asset Management、NVIDIA、Macquarie Capital、Intel Capital、Align Ventures、Tamarack Global、LG Technology Ventures、Salesforce、T-Mobile Ventures、Qualcomm Ventures 参画）、valuation 390億ドル。Figure 03 を2025年10月発表、2026年末まで限定家庭パイロット、価格目標約20,000ドル、BotQ 工場で12,000台/年→100,000台/年へスケール計画。
- **1X Technologies NEO**：2025年10月28日商用ローンチ、Early Access $20,000（2026年配送）、月額サブスク $499/月オプション。1X 独自モデル Redwood（160M params VLT）+ 1X World Model、加えて NVIDIA GR00T N1 を NEO Gamma に「最小限の post-training データで」展開（Bernt Børnich, GTC 2025）。2025年12月 EQT との合意で2026–2030年に最大10,000台を産業展開。
- **Sanctuary AI Phoenix**：Gen7（2024年4月）、Gen8（2024年12月）。20–21 DOF 液圧ハンド、触覚感度 5 mN、25 kg ペイロード、新タスクを <24時間で学習。Magna でパイロット（2024年）、Canadian retailer Mark's で1週間のパイロットで 110 retail-related tasks を完了。
- **Unitree G1 / H1**：G1 ベース価格 $16,000、H1 ベース $90,000（Robot Report, 2024年5月）。ただし G1 は身長127cm/ペイロード2kgで「人間労働の置換は実用上不可」と評価される。
- **Tesla Optimus**：Musk は2025年初に「年内10,000台」を予測したが、2026年1月時点でゼロ。Q1 2026 earnings call で Musk：「Optimus 3 unveil を生産時期にあわせる、量産開始は2026年7–8月頃を想定。」Optimus が「Tesla 外で useful になる」のは2027年以降の見込み。第1世代ライン年産100万台を計画と表明。
- **Boston Dynamics Stretch**：DHL Group が2025年5月13日に MoU 更新、追加1,000台導入を約束（DHL 公式リリース：「the agreement paves the way for the global deployment of more than 1,000 additional units」）。同リリースで「To date, the deployments of Stretch have achieved case unloading rates of up to 700 cases per hour」と性能開示。Boston Dynamics CEO Robert Playter は「Stretch is ideally suited to be the first multi-use case, box picking robot that can meet the diverse needs of DHL」とコメント。

#### 4.3 導入のボトルネック総括

1. **コスト/稼働率**：Walmart 撤退に象徴される ROI 不成立。日本のコンビニ補充は「人件費の高さ×24時間操業」で経済性が成立しやすい。
2. **SKU 多様性**：コンビニ1店舗 ~3,000 SKU、スーパー 30,000+ SKU。ハンド/把持点モデルのカバレッジが課題。
3. **安全規制と人共存**：売場での顧客の前で稼働するハードルが高く、現状は店舗閉店時間帯・バックヤードに限定が多い（Simbe Tally は例外的に客在中でも稼働）。
4. **データ収集の戦略**：Telexistence の motion dataset service、Figure × Brookfield、TX × SEJ の20,000店舗のように、「データ独占」が次の競争軸。
5. **ロングテール失敗のリカバリ**：TX Ghost が示すように、自律98%+遠隔2%の構図でも経済合理性が成立。完全自律99.9%は不要かもしれない、というラディカルな実装思想が日本発で確立しつつある。

## Recommendations

### 短期（~12ヶ月）で取り組むべきこと

1. **対象を絞った Pick-and-Place 自動化から始める**：PETボトル・缶・カップ麺・パウチなど形状が安定した SKU を最初に。Telexistence の TX SCARA/Ghost の98%自律+遠隔リカバリ・モデルが現状最強のリファレンスアーキテクチャ。
2. **ハンドはハイブリッド設計**：吸着パッド（中央）+並列指/3指（把握補強）+力制御を基本とし、食品グレード認証（IP69K、FDA、日本食品衛生法）を満たす Schmalz mGrip 系を有力選択肢に。
3. **把持点検出は AnyGrasp + Contact-GraspNet ベースライン**を起点に、自社データで fine-tune。GraspGen も評価候補。

### 中期（1–3年）

4. **VLA 基盤モデル採用**：Physical Intelligence π0.5（openpi で公開）、NVIDIA GR00T N1（GitHub/Hugging Face で公開）を post-training base にし、自社実機データを用いた数千時間規模の遠隔操作デモで微調整。
5. **データ収集戦略の構築**：teleoperation（VR/exoskeleton）+ ego-centric ヒューマン動画 + 合成データ（Isaac Sim/Cosmos）の3段ピラミッドを確立。日本の事業者は AIRoA や Telexistence の dataset service 利用を検討。
6. **モーション計画スタック**：MoveIt 2 + cuRobo（NVIDIA）+ TrajOpt のハイブリッドが現状最も実用的。リアクティブ層に diffusion policy/π系を載せる。

### 長期（3–5年）、変更トリガとなるベンチマーク

7. **Humanoid 検討開始の閾値**：Apptronik Apollo / Figure 03 / 1X NEO が以下を満たした時点で本格評価：
   - 1日あたり 8時間連続稼働の MTBF が >200時間
   - 未知 SKU での Pick-and-Place 成功率 >97%（独立評価）
   - 1台あたり TCO が3年間で人件費の60%以下
8. **VLA 採用切替の閾値**：単一ポリシーが店舗 SKU 全カテゴリ（剛体・変形・透明・反射）で >95% 成功、未知棚配置で zero-shot >85% を達成した時点で内製スタックから VLA 一本化へ移行。
9. **撤退判断基準**：Walmart×Bossa Nova の教訓——shelf-scan だけで補充実行を伴わない単機能ロボットは ROI が出ない。把持+陳列まで一気通貫の完結機能を最低要件とすること。

## Caveats（限界と注意）

- **VLA の "汎化性" は段階的**：π0.5、GR00T N1、Gemini Robotics 1.5 などの公開デモは厳選環境であり、商用 SLA に直結する独立第三者ベンチマークは2026年5月時点で確立していない。Figure の barcode 成功率 94.4% も自社報告。
- **ヒューマノイドの店舗導入は2026年5月時点でほぼゼロ**：倉庫（Digit×GXO）はリファレンスがあるが、客在中の売場フロアでの ablation 公開実証は無い。SEJ×Telexistence Astra は2029年が公式目標。
- **Tesla Optimus の進捗は遅延**：Musk 自身が2026年1月に「ゼロが useful work」を認めた。第三者導入は2027年以降の見通し。
- **Bossa Nova / Marty の歴史**：棚スキャン単機能ロボの ROI は限定的という米国の実例。日本でも shelf-scan のみのモデルは KDDI×ローソンの実証で「品出しまで」拡張されている点に注意。
- **市場予測は割引が必要**：Global Market Insights のソフトグリッパ CAGR 35.1% など、市場予測は導入加速のシナリオ前提。実装は SKU/業態ごとに個別判断が必要。
- **データソースの偏り**：本レポートでは英語圏（arXiv/RobotReport/PR Newswire）と日本語企業発表に依存しており、中国系（Unitree、UBTECH、Fourier 等）の店舗実証情報は限定的。Unitree G1 の身体性が小売現場で実用に足るかは2026年5月時点で疑問符付き。
